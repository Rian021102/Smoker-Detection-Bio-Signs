# Identify Smoker Status Using Bio - Signs

## Project Description
In this project:
1. I drew insight from EDA to see what parameters affected by smoking
2. I develop a Machine Learning Prediction to identify smoker status by their bio-signs such as blood pressure, weight, waist measurement, dental carries etc.
## Project Background
Smoking has been proven to have negative effect on individiual health. According to a World Health Organization report, the number of deaths caused by smoking will reach 10 million by 2030. Scientists have recorded bio-signs from more than 30,000 individual who smoke and don't such as age, weight, cholesterol level, blood pressure, fasting blood sugar etc to identify the smoker status. Based on this record, we develop a machine learning prediction to identify smoker status by each individual bio-signs
